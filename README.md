# Public Skills

Reusable agent skills. This repo currently has one skill:

- `mcmaster-web-performance`: build clean, fast websites with an app shell,
  inline critical CSS, versioned assets, prefetch-on-hover navigation, and a
  Lighthouse 100 checklist.

## Install

Copy `mcmaster-web-performance` into the place your agent reads skills.

| Agent | Drop the folder here |
| --- | --- |
| Codex user skill | `~/.agents/skills/mcmaster-web-performance` |
| Codex project skill | `<project>/.agents/skills/mcmaster-web-performance` |
| Claude Code user skill | `~/.claude/skills/mcmaster-web-performance` |
| Claude Code project skill | `<project>/.claude/skills/mcmaster-web-performance` |
| Claude.ai custom skill | Upload a ZIP of `mcmaster-web-performance` |
| Other agents | Use the folder wherever your agent expects `SKILL.md` |

For Claude.ai, the ZIP should open directly to `SKILL.md`, not to an extra parent
folder.
